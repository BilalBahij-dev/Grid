01 / Introduction

What Is CSS Grid?

CSS Grid is a two-dimensional layout system. You set a container to display: grid and its children become grid items placed into rows and columns simultaneously — unlike Flexbox which works in one axis at a time.

🏛️

Grid Container

The parent element. Apply display: grid here to turn direct children into grid items.

🧩

Grid Items

Direct children of the grid container. They sit inside cells formed by the intersection of rows and columns.

➡️

Grid Tracks

The rows and columns you define. Each track is the space between two adjacent grid lines — either a row track or a column track.

⬛

Grid Cell

The intersection of one row track and one column track. It's the smallest unit of a grid — like a table cell.

🔲

Grid Area

One or more cells grouped together. Items can span multiple cells using grid-column and grid-row.

📐

Grid Lines

The dividing lines that make up the grid structure. They are numbered starting from 1. You can place items by referencing these line numbers.

.item {
  grid-column: 1 / 3;    /* from line 1 to line 3 (spans 2 cols)  */

06 / Property

grid-template-areas

Name regions of your grid using ASCII-art strings. Each string is a row; each word is a cell's name. Items are then placed by setting grid-area to the matching name. Use . for empty cells.

Classic page layout: header, sidebar, main, footer grid-template-areas

Magazine layout with empty cell (dot) . = empty cell

.container {
  display: grid;
  grid-template-columns: 160px 1fr;
  grid-template-rows: 60px 1fr 50px;
  grid-template-areas:
    "header  header"
    "sidebar main"
    "footer  footer";
}

/* Place items by name — no line numbers needed! */
.header  { grid-area: header;  }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main;    }
.footer  { grid-area: footer;  }

/* Use a dot ( . ) to leave a cell empty */
grid-template-areas:
  "hero hero aside"
  ".    body aside"; /* dot = empty cell */

10 / Grid Functions & Units

repeat() · fr · minmax()

CSS Grid introduces powerful functions and units that make responsive layouts simple without any media queries. These are the true superpowers of Grid.

Feature What it does Default
fr unit Stands for "fraction of the available free space". Works like flex-grow. 1fr 2fr = first column gets ⅓, second gets ⅔ of space. —
repeat() Shorthand to avoid repetition: repeat(4, 1fr) = 1fr 1fr 1fr 1fr. Works with auto-fill and auto-fit for truly responsive grids. —
minmax() Sets a size range: minmax(200px, 1fr) = minimum 200px, maximum 1fr. Used with auto-fill for responsive grids that need no media queries. —

/* fr unit — fraction of remaining free space */
grid-template-columns: 1fr 1fr 1fr;          /* 3 equal cols             */
grid-template-columns: 1fr 2fr;               /* 1/3 then 2/3             */
grid-template-columns: 200px 1fr;             /* fixed sidebar + flex main */

/* repeat() — shorthand for repeated values */
grid-template-columns: repeat(4, 1fr);        /* 4 equal cols             */
grid-template-columns: repeat(3, 100px 1fr); /* alternating pattern      */

/* minmax(min, max) — responsive sizing */
grid-template-columns: minmax(200px, 1fr);    /* min 200px, grow to 1fr   */
grid-template-rows:    minmax(100px, auto);   /* min 100px, grows w/ content */

/* The holy grail — fully responsive, no media queries */
grid-template-columns: repeat(auto-fit, minmax(240px, 1fr));
/* auto-fill = create as many tracks as fit (may leave empty cols) */
/* auto-fit  = collapse empty tracks (items stretch to fill)       */
